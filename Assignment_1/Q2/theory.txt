Q2:- Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML documents
Ans:- In HTML, comments work as a way to include non-visible notes or explanations about the particular code that will be easy for the developer to understand that code or when another developer try to look over the code  and he can get to know about that particular code . Comments are ignored by web browsers and are meant for human readers and developers. making it easier for developers to understand, maintain, and collaborate on HTML documents.
The syntax for creating comments in HTML is simple and consists of <!-- to start a comment and --> to end it. Anything between these delimiters is treated as a comment and is not rendered or executed by the web browser.
